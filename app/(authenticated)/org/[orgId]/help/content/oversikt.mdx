### Förstå översiktsdashboarden

Översiktssidan ger dig en snabb överblick över alla rapporter i organisationen. Här hittar du nyckeltal, trender och visualiseringar av data.

#### Nyckeltal

Överst på sidan visas fyra viktiga nyckeltal:

- **Totalt antal rapporter** - Hur många rapporter som skapats totalt
- **Genomsnittliga dagar** - Genomsnittligt antal dagar per rapport
- **Maxdagar** - Den längsta tiden en rapport har tagit
- **Aktiva kategorier** - Antal problemkategorier som används

<HelpImagePlaceholder caption="Nyckeltalskort med statistik" />

#### Dagstrender

Trenddiagrammen visar hur rapporter och problem utvecklas över tid:

- **Dagliga trender** - Visar data per dag
- **Veckovisa trender** - Aggregerad data per vecka
- **Månadsvisa trender** - Aggregerad data per månad

<HelpTip variant="tip">
**Tips:** Hovra över diagrammen för att se exakta värden för varje datapunkt.
</HelpTip>

#### Kategorifördelning

Ett cirkeldiagram visar hur rapporter fördelar sig mellan olika problemkategorier. Detta hjälper dig att se vilka typer av problem som är vanligast.

<HelpImagePlaceholder caption="Cirkeldiagram över kategorifördelning" />

#### Teknikerprestation

Ett stapeldiagram visar hur många rapporter varje tekniker har hanterat. Detta ger en överblick över arbetsfördelningen.

#### Toppproblem

En lista över de vanligaste problemen baserat på rapporterade ärenden. Detta hjälper dig att identifiera återkommande problem som kanske behöver åtgärdas.

#### Interaktiva diagram

Många av diagrammen är interaktiva:

- Hovra över datapunkter för att se detaljer
- Klicka på legendförklaringar för att dölja/visa dataserier
- Vissa diagram kan zoomas för att se detaljer
